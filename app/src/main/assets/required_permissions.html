Required Permissions

This app uses location information to calculate all the information shown in this app, this app does not collect any user data and does not sell any information of the user to any third party. These are the permissions the app uses to show the meaningful information to the user.


Compulsory Permissions

This permissions are required and application's most of the feature and functions depends on it, denying them will break some of the features bt application will continue to work. A prompt will appear asking for location permission after pressing grant button below.

  • ACCESS_FINE_LOCATION

    • It is required to run the GPS of the device and fetch the possibly accurate location details, this app uses default device's location API and this permission is required to utilise that API.

  • ACCESS_COARSE_LOCATION

    • Used to show a precise location data in case GPS is unavailable or until available.



    Automatic Permissions

    These permissions are granted automatically by the system.

  • INTERNET

    • Used to fetch the address of the given location, load map and fetch latitude and longitude using the address in specified location

  • ACCESS_WIFI_STATE

    • Used to check if WiFi connectivity is available.

  • ACCESS_NETWORK_STATE

    • Used to check if network is available and is connected.
