Build86

Status: In development

Last Updated: 24 April, 2021



I have planned a road map for the Build86 and here are the things that I would like to add in this version

• Different font styles for the whole app
• Option to use of Rotation Vector for compass
• Make the app full screen
• New custom bottom navigation style to make the app more new panels friendly in future
• Possibly allow rotation of the app by default


These features are quite tempting and can make the app even more appealing and user friendly but implementing them may take time. Any possible bug reports should be posted on GitHub or via Email and will be added to the queue until the build is finalized.

Added home screen widget for Sun Times and it uses both 12 and 24 hour time formats and these configurations can be changed from the clock menu since this panel is directly related with the clock panel. It will be available in both versions.

Added option to adjust physical properties of the compass based on the real physics formulas to create life-like compass movements and removed the older sensor speed option.
