Change Logs



Added home screen widget for Sun Times and it uses both 12 and 24 hour time formats and these configurations can be changed from the clock menu since this panel is directly related with the clock panel. It will be available in both versions.

Added option to adjust physical properties of the compass based on the real physics formulas to create life-like compass movements and removed the older sensor speed option.

Added art frame sun time widget for additional decoration if needed.

Removed material toolbar from every panel in favour of making the app translucent and add a transparent status bar feature and create a better simplified light interface.

Lite version's apk is now smaller in size and uses a different art for splash screen in favour of making it a sustainable standalone application. Even if it is free it should look good to the eyes.

Added two new physics based motion type for clock needle and random initial animations based on the needle position and rotation angle.

Development Status removed from the settings due to its unnecessary nature.

Added option to toggle seconds precision in Clock panel.

Added toggle to disable the use of Physical Properties in Compass panel as fallback mechanism for devices that suffer from compass disappearing issue due to the implementation of newer compass framework that mimics life-like compass rotation.

Added two new splash screen arts.

Added widget for moon illumination.

Added accent colors into the whole app.
