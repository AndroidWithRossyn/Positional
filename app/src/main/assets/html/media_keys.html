Using Media Keys

The GPS panel has an option to override default media keys function from the device and use them to few features of the app. The different use cases are mentioned below.

These button events will only work if the Use Media Keys option is toggled on from the GPS Menu

  • Volume Up

    • This will zoom in the currently visible location on the map. This option will zoom-in the center of the map regardless of the marker's position on the screen. For centering the marker use Location on the header panel.

  • Volume Down

    • Same behavior as Volume Up key except it will zoom out the map.

  • Play/Pause

    • Such buttons are available via external media devices such as Bluetooth controller. Pressing this key will move the map's camera to the marker meaning the marker or the current location will be centered on the display panel.
